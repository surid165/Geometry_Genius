1.DOM & Its Purpose

DOM stands for Document Object Model.DOM is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.The purpose of DOM is to provide a way to interact and manipulate web pages Dynamically.


2.How to select elements from DOM

The most common methods to select elements is to use "document" and the method name.
1. getElementById: This method is used to select an element by its unique id attribute.
2. getElementsByClassName: This method is used to select elements by their class names. It returns a array-like of elements with the specified class.
3. getElementsByTagName: This method selects elements by their HTML tag name and returns a collection of elements.
4. querySelector: This method selects element using CSS-style selectors. It returns the first matching element.


3.Event Bubble

Event bubbling is a technique by which when an event is triggered on a nested element it bubbles up through the parent nodes in the DOM tree, it calls the event handlers on those parent elements in turn.


4.Callback Function

A callback function is a function that is passed as an argument to another function.It is executed after the completion of that function. Callback functions are commonly used in JavaScript and other programming languages for various purposes. They are essential in Asynchronous programming,event handling,Error handling, and many other cases.
